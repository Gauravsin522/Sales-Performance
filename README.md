Sales Performance Dashboard
This project presents a Sales Performance Dashboard built in Excel to analyze overall sales, quantity sold, average discount, and category-wise performance.

The dashboard includes:

Monthly sales trend analysis.

Sales performance by sales representative.

Sales split by payment method.

Product category-wise sales comparison.

Interactive filters for product category and region.

It provides a clear view of business performance and helps identify sales patterns and key areas for improvement.
